# Public Assets

Stavite sve statičke fajlove ovde.

## Logo

Dodajte **logo.png** direktno u ovaj folder:

```
public/logo.png
```

### Preporuke za logo:
- Format: PNG sa transparentnom pozadinom
- Dimenzije: 512x512 piksela ili veće (kvadratni format)
- Veličina fajla: < 500KB

Logo će biti prikazan:
- Na login ekranu (centralno)
- U header-u dashboard-a (gore levo)
